📖 Was ist eine Stellungnahme?
Argumentativ schreiben — These, Argumente, Gegenargumente, Fazit.
Ziel: Eine Stellungnahme bezieht eine klare Position zu einer Frage, untermauert diese mit mindestens zwei Argumenten und setzt sich sachlich mit einer Gegenposition auseinander — abgeschlossen durch ein überzeugendes Fazit.

🏗️ Die vier Bausteine einer Stellungnahme

  • Einleitung mit These: Das Thema wird knapp eingeführt, dann folgt die eigene Position in einem einzigen klaren Satz. Die These steht idealerweise am Ende des ersten Absatzes.
  • Argumente mit Belegen: Zwei bis drei Argumente werden ausgeführt, jedes mit einem konkreten Beispiel, einer Statistik oder einer Erfahrung unterlegt. Das stärkste Argument kommt zuletzt.
  • Gegenposition und Entkräftung: Ein wichtiger Einwand wird ernst genommen und anschließend sachlich entkräftet („Zwar … allerdings …“). Das zeigt differenziertes Denken.
  • Fazit mit Empfehlung: Die eigene Position wird noch einmal bekräftigt, ergänzt durch eine Forderung, einen Appell oder einen Ausblick.

🎯 Merkmale einer überzeugenden Stellungnahme

  • Klare Position: Die Leserin erkennt sofort, wofür oder wogegen argumentiert wird — kein „vielleicht“, kein „ein bisschen“.
  • Logische Argumentstruktur: Jedes Argument folgt dem Muster „Behauptung – Begründung – Beispiel“.
  • Sachlicher Ton: Auch bei emotionalen Themen bleibt der Text ruhig, präzise und begründet. Keine Polemik, keine Beschimpfung der Gegenseite.
  • Differenzierung: Gegenargumente werden ernst genommen, nicht ignoriert.
  • Verbindende Konnektoren: Zwar … aber …; einerseits … andererseits …; dennoch; allerdings; folglich; infolgedessen.
  • Konjunktiv II: Für hypothetische Folgen und Empfehlungen: „Eine Regelung würde bedeuten …“, „Es wäre sinnvoll, wenn …“

⚠️ Häufige Fehler

  • Keine klare These: Der Text schwimmt, weil die Position unklar bleibt.
  • Behauptung ohne Beispiel: „Das ist gefährlich“ überzeugt nicht — ein konkretes Beispiel schon.
  • Emotionale Angriffe auf die Gegenposition („So etwas ist lächerlich“) untergraben die eigene Glaubwürdigkeit.
  • Gegenargumente fehlen ganz — die Stellungnahme wirkt einseitig.
  • Das Fazit wiederholt nur die Einleitung, ohne neuen Impuls (Forderung, Ausblick).
💬 Redemittel für Stellungnahmen
Bausteine, die Argumentation strukturieren und sprachlich variieren lassen.
📌 Einleitung und Thema nennen
  • In letzter Zeit wird zunehmend diskutiert, ob …
  • Die Frage, ob …, wird derzeit kontrovers debattiert.
  • In den Medien ist immer häufiger von … die Rede.
  • Das Thema … beschäftigt die Öffentlichkeit seit …
  • Der folgende Text nimmt Stellung zu der Frage, ob …
🎯 Eigene Position formulieren
  • Meiner Ansicht nach …
  • Meines Erachtens sollte …
  • Ich vertrete die Auffassung, dass …
  • Ich bin fest davon überzeugt, dass …
  • Aus meiner Sicht spricht vieles dafür, dass …
  • Für mich steht fest, dass …
➕ Argumente strukturieren
  • Ein erstes wichtiges Argument ist, dass …
  • Zunächst ist darauf hinzuweisen, dass …
  • Darüber hinaus / Zusätzlich gilt zu bedenken, dass …
  • Ein weiterer zentraler Aspekt betrifft …
  • Nicht zuletzt spielt … eine entscheidende Rolle.
  • Besonders schwer wiegt das Argument, dass …
🔎 Beispiele und Belege anführen
  • Ein konkretes Beispiel dafür ist …
  • Dies lässt sich anhand folgender Zahlen verdeutlichen: …
  • Eine aktuelle Studie belegt, dass …
  • So zeigt etwa die Erfahrung, dass …
  • Man denke nur an …
  • Exemplarisch sei hier … genannt.
⚖️ Gegenargumente aufgreifen und entkräften
  • Zwar wird häufig eingewendet, dass … Allerdings …
  • Einerseits … andererseits …
  • Es ließe sich zwar argumentieren, dass … Dem ist jedoch entgegenzuhalten, dass …
  • Kritiker führen oft an, dass … Dies trifft jedoch nur bedingt zu, weil …
  • Gegen diese Position spricht allerdings, dass …
  • Diesem Einwand ist zu entgegnen, dass …
🎬 Fazit und Schlussfolgerung
  • Zusammenfassend lässt sich feststellen, dass …
  • Alles in allem überwiegen die Argumente, die für … sprechen.
  • Aus den genannten Gründen plädiere ich für …
  • Es bleibt festzuhalten, dass …
  • Daher sollte dringend …
  • Ein Umdenken wäre an dieser Stelle dringend geboten.
📄 Modelltext: Homeoffice gesetzlich regeln?
Ein Mustertext in vier Teilen — These, Argumente, Gegenposition, Fazit.
Stellungnahme zur Frage: Sollte das Recht auf Homeoffice gesetzlich verankert werden?

1. Einleitung und These

Seit der Corona-Pandemie hat das Homeoffice in Deutschland einen festen Platz im Arbeitsalltag gefunden. Dennoch ist die rechtliche Lage weiterhin unklar: Arbeitgeber können Heimarbeit gewähren, müssen es aber nicht. Meiner Ansicht nach sollte ein gesetzlicher Anspruch auf Homeoffice eingeführt werden, wo die Tätigkeit es zulässt — aus Gründen der Fairness, der Produktivität und des Klimaschutzes.

2. Argumente

Ein erstes wichtiges Argument ist die Vereinbarkeit von Beruf und Familie. Eltern, Pflegende und Menschen mit langen Pendelwegen profitieren erheblich von flexibleren Arbeitsorten. Eine Studie der Hans-Böckler-Stiftung zeigt, dass Beschäftigte im Homeoffice durchschnittlich zufriedener mit ihrem Lebensalltag sind als reine Büroarbeitende.

Darüber hinaus spielt die Produktivität eine entscheidende Rolle. Zahlreiche Unternehmen berichten, dass ihre Mitarbeitenden zu Hause konzentrierter arbeiten, weil Großraumbüros und ungeplante Unterbrechungen wegfallen. Nicht zuletzt wirkt sich das Homeoffice positiv auf das Klima aus: Weniger Pendelverkehr bedeutet weniger CO₂-Emissionen.

3. Gegenposition und Entkräftung

Zwar wird häufig eingewendet, dass ein gesetzlicher Anspruch die unternehmerische Freiheit einschränke und für kleinere Betriebe kaum umsetzbar sei. Allerdings ließe sich diesem Einwand mit einer differenzierten Regelung begegnen: Der Anspruch würde nur dort gelten, wo die Tätigkeit dies zulässt, und Ausnahmen für besondere Härtefälle wären selbstverständlich vorgesehen.

4. Fazit

Zusammenfassend lässt sich feststellen, dass die Vorteile einer gesetzlichen Regelung deutlich überwiegen. Aus den genannten Gründen plädiere ich für einen maßvollen Rechtsanspruch mit klaren Ausnahmen. Ein solches Gesetz würde der modernen Arbeitswelt Rechnung tragen und zugleich Arbeitnehmende wie Arbeitgeber schützen.

💡 Beobachte: Der Text bezieht sofort Position („sollte … eingeführt werden“), stützt sie mit drei konkreten Argumenten, nimmt den wichtigsten Einwand ernst („Zwar … allerdings …“) und endet mit einer klaren Empfehlung. Die gelb markierten Ausdrücke sind Fachbegriffe und Konnektoren, die du in eigenen Stellungnahmen wiederverwenden kannst.

📝 Aufgabenstellung
Schreibe eine Stellungnahme (200–250 Wörter) zu einem der folgenden Themen. Dein Text soll eine klare These enthalten, zwei bis drei Argumente mit Beispielen, eine Auseinandersetzung mit einem Gegenargument und ein Fazit.
  • Sollten KI-Werkzeuge wie ChatGPT an Universitäten erlaubt sein?
  • Sollte die reguläre Wochenarbeitszeit auf 35 Stunden gesenkt werden?
  • Ist die Vier-Tage-Woche ein Modell für die Zukunft?
  • Sollten große Unternehmen verpflichtet werden, ihren CO₂-Fußabdruck jährlich zu veröffentlichen?
  • Sollte das Autofahren in Innenstädten stärker eingeschränkt werden?
  • Brauchen wir ein Wahlrecht ab 16 Jahren?
💡 Tipp: Plane deinen Text vor dem Schreiben in Stichpunkten: These – Argument 1 – Argument 2 – Gegenargument + Entkräftung – Fazit. Dann fällt der Aufbau leicht.

✅ Checkliste für deine Stellungnahme
  • Ist deine These am Ende der Einleitung klar formuliert?
  • Hast du mindestens zwei Argumente mit konkreten Beispielen?
  • Hast du ein Gegenargument aufgegriffen und entkräftet?
  • Endet der Text mit einem Fazit (Zusammenfassung + Forderung oder Ausblick)?
  • Sind die Absätze durch Konnektoren („zwar … allerdings“, „darüber hinaus“, „zusammenfassend“) verbunden?
  • Bleibt der Ton sachlich — ohne Beschimpfungen oder Polemik?
