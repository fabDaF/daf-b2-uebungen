Wechselpräpositionen

B2.2 · Einheit 5 · Grammatik · 2052G

🔍 Entdecken: Was fällt auf?

Lesen Sie die Satzpaare und achten Sie auf die Präposition und den Kasus. Was ist der Unterschied?
Satz ASatz B
Das Buch liegt auf dem Tisch.Ich lege das Buch auf den Tisch.
Die Lampe hängt über dem Tisch.Ich hänge die Lampe über den Tisch.
Das Labor befindet sich im Keller.Wir gehen in den Keller.
Die Probe steht neben dem Mikroskop.Ich stelle die Probe neben das Mikroskop.
Die Daten befinden sich auf dem Server.Ich lade die Daten auf den Server.
Beobachtung: Satz A beschreibt eine Position (Wo?), Satz B beschreibt eine Richtung (Wohin?). Dieselbe Präposition — aber unterschiedliche Kasus!

📋 Die Regel: Wechselpräpositionen

Wechselpräpositionen stehen entweder mit Dativ (Wo? = Position) oder mit Akkusativ (Wohin? = Richtung/Bewegung).
an
am Rand / an die Wand
auf
auf dem Tisch / auf den Tisch
hinter
hinter dem Haus / hinter das Haus
in
im Labor / ins Labor

über

unter

vor
vor dem Eingang / vor den Eingang
zwischen
zwischen den Regalen / zwischen die Regale
FrageKasusBeispiel
Wo? (Position, kein Ortswechsel)DativDas Gerät steht auf dem Labortisch.
Wohin? (Richtung, Bewegung)AkkusativEr stellt das Gerät auf den Labortisch.
Ausnahme: hängen, liegen, stehenDativDas Poster hängt an der Wand.
Ausnahme: hängen, legen, stellenAkkusativEr hängt das Poster an die Wand.
Tipp: Verben wie liegen, stehen, hängen, sitzen, sich befinden → Dativ (Wo?). Verben wie legen, stellen, hängen, setzen, stecken → Akkusativ (Wohin?).

Ergänzen Sie den richtigen Artikel (Dativ oder Akkusativ).

1. Die Probe liegt auf Labortisch. (Position → Dativ)

2. Der Forscher legt die Probe auf Labortisch. (Richtung → Akkusativ)

3. Das Mikroskop steht neben Computer. (Position → Dativ)

4. Sie stellt das Mikroskop neben Computer. (Richtung → Akkusativ)

5. Die Ergebnisse hängen an Wand. (Position → Dativ)

6. Er hängt die Ergebnisse an Wand. (Richtung → Akkusativ)

7. Die Daten befinden sich auf Server des Instituts.

8. Wir laden die Messdaten auf Server des Instituts hoch.

10. Wir gehen in neue Labor im dritten Stock.
