✅ Das Passiv auf einen Blick

  • Vorgangspassiv: werden + Partizip II → beschreibt eine Aktion/einen Prozess
  • Zustandspassiv: sein + Partizip II → beschreibt ein Ergebnis/einen Zustand
  • Passiv mit Modalverben: Modalverb + Partizip II + werden
  • Agens: optional mit von + Dativ → „von der EU gefördert"

📋 Das Passiv — Systematische Übersicht

Im Deutschen gibt es zwei Passivformen. Beide sind im B2-Niveau wichtig.

1. Vorgangspassiv — werden + Partizip II

Beschreibt eine Handlung oder einen Prozess. Das Subjekt ist nicht der Handelnde, sondern das Objekt der Handlung.

werden (konjugiert) + Partizip II  |  optional: von + Dativ (Agens)
Aktiv Die EU fördert das Programm.
Der Ausschuss prüfte die Bewerbungen.
Man vergibt das Stipendium jährlich.
Passiv (Vorgang) Das Programm wird (von der EU) gefördert.
Die Bewerbungen wurden geprüft.
Das Stipendium wird jährlich vergeben.

2. Zustandspassiv — sein + Partizip II

Beschreibt einen Zustand, der das Ergebnis einer abgeschlossenen Handlung ist. Der Prozess ist bereits beendet.

sein (konjugiert) + Partizip II
Das Formular ist bereits ausgefüllt. (Zustand: Es wurde schon ausgefüllt.)
Die Plätze sind vergeben. (Zustand: Es gibt keine freien Plätze mehr.)
Die Unterkunft ist schon gebucht. (Zustand: Die Buchung wurde abgeschlossen.)

3. Passiv mit Modalverben

Das Modalverb wird konjugiert, das Partizip II und der Infinitiv von werden stehen am Ende.

Modalverb + Partizip II + werden
Die Bewerbung muss bis Ende März eingereicht werden.
Das Visum soll rechtzeitig beantragt werden.
Die Unterlagen können online eingereicht werden.
💡 Merkhilfe: Vorgangspassiv = wird gemacht (etwas passiert gerade oder regelmäßig). Zustandspassiv = ist gemacht (es ist bereits fertig). Der Unterschied ist oft entscheidend für die Bedeutung!
