Der Passiversatz mit „sich lassen“

„Sich lassen“ + Infinitiv ist eine Alternative zum Passiv. Sie drückt aus, dass etwas möglich ist oder getan werden kann.

Beispiele:

  • Das lässt sich nicht sagen = Das kann nicht gesagt werden (Passiv: Das kann nicht gesagt werden)
  • Das Problem lässt sich lösen = Das Problem kann gelöst werden
  • Diese Sätze lassen sich umformen = Diese Sätze können umgeformt werden
  • Die Fehler lassen sich vermeiden = Die Fehler können vermieden werden
  • Das lässt sich beweisen = Das kann bewiesen werden
  • Der Text lässt sich verbessern = Der Text kann verbessert werden
