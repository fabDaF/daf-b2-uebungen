Was ist ein Konzessiv- und Konsekutivsatz?

Konzessivsätze zeigen einen Gegensatz: „Obwohl es regnet, gehe ich nach draußen." Konsekutivsätze zeigen eine Folge: „Es regnet so stark, dass ich nicht gehe." Erkenne den Unterschied in den Sätzen!

Schritt 1: Konzessiv vs. Konsequenz

Lese die zwei Sätze:

A: Obwohl der Film schlecht war, hat mir die Zeit Spaß gemacht. (Gegensatz!)

B: Der Film war so langweilig, dass ich einschlafen bin. (Folge!)

Schritt 2: Konjunktionen erkennen

Konzessivsätze nutzen: obwohl, obgleich, wenngleich, auch wenn

Konsekutivsätze nutzen: sodass, so ... dass, derart ... dass

Welche Konjunktion passt hier? „Er war krank, _____ er nicht zur Arbeit ging."

Schritt 4: Anwendungsfall

Welcher Satz ist ein Konzessivsatz?

Regel: Konzessiv- und Konsekutivsätze

  • Konzessivsatz: Zeigt einen Gegensatz. „Obwohl / obgleich / wenngleich / auch wenn + Nebensatz"
  • Konsekutivsatz: Zeigt eine Folge. „Sodass / so ... dass / derart ... dass + Nebensatz"
  • Beispiel Konzessiv: Obwohl ich müde bin, arbeite ich weiter.
  • Beispiel Konsequenz: Ich bin so müde, dass ich schlafen muss.
Regel: Konzessivsätze drücken einen Gegensatz aus, Konsekutivsätze zeigen eine Folge oder Konsequenz.

Konzessivsätze (Gegensatz)

Konjunktionen: obwohl, obgleich, wenngleich, auch wenn, trotzdem

Struktur: Obwohl + Nebensatz, Hauptsatz

Beispiel: Obwohl es teuer ist, kaufe ich das Smartphone.

Konsekutivsätze (Folge)

Konjunktionen: sodass, so ... dass, derart ... dass

Struktur: Hauptsatz + sodass / so ... dass + Nebensatz

Beispiel: Das Essen war so schlecht, dass ich es nicht essen konnte.
