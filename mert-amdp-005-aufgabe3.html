Die Aufgabe
Erklären Sie den Unterschied zwischen „gesperrt“ und „Gedankenabreißen“ (#24). Warum ist dies ein SF-Merkmal? (40–60 Wörter)
Originaltext — unverändert

Im Gespräch gibt es plötzlicher Abbruch eines flüssigen Gedankenganges, und gleichzeitig gibt es kein Grund dafür. Das nennt man gesperrt. Es könnte von Außen auch bemerkt. Deshalb ist dies ein SF-Merkmal. Das Gedankenabreißen wird es selbst beurteilt. Der Betroffene erlebt abrupten Abbruch und kann nicht mehr vorherigen Gedanken erinnern.

Satz 1 Im Gespräch gibt es plötzlicher Abbruch einen plötzlichen Abbruch eines flüssigen Gedankengangs, und gleichzeitig gibt es kein keinen Grund dafür.
Ein Lehrstück zu diesem Satz
„es gibt“ + Akkusativ
Nach „es gibt“ steht das Hauptwort im Akkusativ. Aus „ein/kein“ wird dann „einen/keinen“: einen Abbruch, keinen Grund.
Es gibt einen plötzlichen Abbruch. Es gibt keinen Grund dafür. ein → einen, kein → keinen.
Übung — Akkusativ nach „es gibt“
Im Gespräch gibt es plötzlichen Abbruch.
Es gibt Grund dafür.
✓ Richtig — „es gibt einen / keinen …“.
Satz 2 Das nennt man gesperrt.
Dieser Satz ist sprachlich richtig — kurz und klar.
Satz 3 Es könnte von Außen außen auch bemerkt werden.
Zwei Lehrstücke zu diesem Satz
könnte … bemerkt werden
„können/könnte“ zusammen mit einem Passiv braucht am Ende „werden“: Es könnte … bemerkt werden.
Es könnte von außen bemerkt werden. Der Abbruch kann beobachtet werden. Modalverb + Partizip + werden.
Übung — „werden“ ergänzen
Es könnte von außen bemerkt .
Das kann leicht beobachtet .
✓ Richtig — „… bemerkt/beobachtet werden“.
von außen — klein
„außen“ ist ein Umstandswort (Adverb) und wird kleingeschrieben: von außen, nach außen.
von außen sichtbar nach außen ruhig innen / außen → immer klein.
Übung — klein schreiben
Der Abbruch ist von sichtbar.
Sie wirkt nach ruhig.
✓ Richtig — „von/nach außen“ kleingeschrieben.
Satz 4 Deshalb ist dies ein SF-Merkmal.
Dieser Satz ist sprachlich richtig.
Satz 5 Das Gedankenabreißen wird es selbst beurteilt.
Ein Lehrstück zu diesem Satz
nur ein Subjekt
„Das Gedankenabreißen“ ist schon das Subjekt. Ein zweites „es“ ist zu viel: Das Gedankenabreißen wird selbst beurteilt.
Das Gedankenabreißen wird selbst beurteilt. Das Symptom wird selbst beurteilt. Ein Subjekt reicht — kein extra „es“.
Übung — ein Wort einsetzen
Das Gedankenabreißen wird beurteilt.
Das Symptom wird beurteilt.
✓ Richtig — kein zusätzliches „es“.
Satz 6 Der Betroffene erlebt einen abrupten Abbruch und kann sich nicht mehr an den vorherigen Gedanken erinnern.
Ein Lehrstück zu diesem Satz
sich erinnern an + Akkusativ
Das Verb heißt „sich an etwas erinnern“. Es braucht „sich“ und danach „an“ mit dem Akkusativ: sich an den Gedanken erinnern.
Er kann sich an den Gedanken erinnern. Ich erinnere mich an das Gespräch. sich + an + Akkusativ.
Übung — „sich … an“ ergänzen
Er kann nicht mehr Gedanken erinnern.
Ich erinnere an das Gespräch.
✓ Richtig — „sich an … (Akkusativ) erinnern“.
